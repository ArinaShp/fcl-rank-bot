# FCL Management v6.4

В публичные уведомления встроены фирменные баннеры The Faceless Ones.

## Баннеры используются для

- повышения и назначения ранга;
- достижения;
- годовщины;
- хроники семьи;
- еженедельной сводки;
- назначения Полномочного Посла;
- дипломатического паспорта союза;
- доски почёта.

Все изображения находятся в папке `assets`. Её необходимо загрузить в репозиторий вместе с `bot.py`.

Бот прикрепляет изображение к сообщению и показывает его внутри embed. Если файл отсутствует, уведомление всё равно отправляется, но без баннера.
