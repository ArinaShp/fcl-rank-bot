# FCL Management v6.2

Обновлённая система достижений The Faceless Ones.

## Автоматические достижения

### Первый шаг

Выдаётся один раз после первого получения роли `Призванный`.

- сохраняется в личном деле;
- публикуется в канале доски почёта;
- даёт 0 баллов.

### Ветеран семьи

Выдаётся через 180 дней с момента первого допуска.

- сохраняется в личном деле;
- публикуется в канале доски почёта;
- даёт 3 балла;
- автоматически выдаёт роль `Ветеран`.

Название роли можно изменить через переменную:

```text
VETERAN_ROLE_NAME
```

## Команда /достижение

Теперь нужно выбрать только:

```text
участник
достижение
```

Название, описание и количество баллов бот подставляет самостоятельно.

## Ручные достижения

- Знак доверия — 2 балла
- Опора семьи — 2 балла
- Безупречная служба — 2 балла
- След в истории — 3 балла
- Участник операции — 1 балл
- Ключевой участник — 3 балла
- Организатор — 1 балл
- Наставник — 1 балл
- Хранитель традиций — 3 балла
- Голос союза — 2 балла
- Укрепление союза — 2 балла

Одно и то же достижение не выдаётся одному участнику повторно.
