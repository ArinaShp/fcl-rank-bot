# FCL Management v6.1

Финальная полировка внутренней системы The Faceless Ones.

## Без засорения личных сообщений

Бот больше не пишет пользователям в личные сообщения:
- при повышении;
- при понижении;
- при исключении;
- после допуска в семью;
- после допуска к Посольству.

Пользователь получает только скрытое подтверждение внутри Discord после выполненного действия.

## Новое

- `/мой-профиль` — личное дело самого участника;
- публичные оформленные объявления о повышениях;
- автоматические годовщины: 30, 90, 180, 365 дней и далее ежегодно;
- доска почёта может публиковаться в отдельный канал.

ANNOUNCEMENT_CHANNEL_ID
