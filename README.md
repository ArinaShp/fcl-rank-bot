# FCL Management Bot v4

Команды:

- `/управление` — основная кадровая панель;
- `/карточка` — карточка участника;
- `/история` — последние изменения ранга;
- `/состав` — количество участников по рангам;
- `/исключить` — отдельная команда исключения.

Панель `/управление` содержит кнопки:

- Повысить;
- Понизить;
- Назначить ранг;
- История;
- Исключить;
- Обновить.

Бот автоматически снимает старый ранг, выдаёт новый, пишет кадровый журнал и сохраняет историю в SQLite.

## Переменные Render

```text
DISCORD_TOKEN=токен бота
GUILD_ID=ID сервера
ROLE_CHANNEL_ID=ID закрытого канала управления
LOG_CHANNEL_ID=ID канала журнала
DATABASE_PATH=fcl_management.db
```

## Обновление

1. Замени файлы в локальной папке репозитория.
2. В GitHub Desktop сделай commit.
3. Нажми Push origin.
4. Render автоматически развернёт новую версию.

Важно: на бесплатном Render локальная SQLite-база может потеряться при пересоздании инстанса. Для постоянной истории позже лучше подключить PostgreSQL.
