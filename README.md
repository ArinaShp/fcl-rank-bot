# FCL Management v6.3

Упрощённый дипломатический модуль The Faceless Ones.

## Паспорт союза

Команда:

```text
/союз
```

Содержит только:

- название семьи;
- статус;
- основной представитель;
- краткое описание.

Доступные статусы:

- Действующий;
- Приостановлен;
- Завершён.

## Полномочный Посол

Команда:

```text
/назначить-посла
```

Выдаёт лидеру союзной семьи роль:

```text
Полномочный Посол
```

Назначение публикуется в журнале дипломатии.

Название роли можно изменить через переменную Render:

```text
PLENIPOTENTIARY_ROLE_NAME
```

## Удалено

Команда `/совместная-операция` полностью удалена.
