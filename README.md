# FCL Management v5.5

Цельная версия оформления допуска в стиле The Faceless Ones.

## Команда

```text
/создать-доступ
```

В параметр `баннер` можно загрузить изображение.

## Что происходит после нажатия

1. Участник подтверждает Static ID.
2. Бот выдаёт роль `Призванный`.
3. Никнейм меняется на `Призванный | Static ID`.
4. Событие сохраняется в базе.
5. Администрация получает запись в журнал.
6. Участник получает личное приветствие.
7. При последующей смене ранга бот обновляет обозначение автоматически.

Тексты сообщения, подтверждения, личного приветствия и журнала оформлены единообразно и без лишней технической информации.
