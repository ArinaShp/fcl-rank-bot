# FCL Rank Bot

Небольшой бот для Discord-сервера **The Faceless Ones**.

Команда:

```text
/ранг участник новый_ранг
```

Бот автоматически:

1. находит выбранного участника;
2. снимает с него все прежние ранговые роли;
3. выдаёт один выбранный ранг;
4. при необходимости пишет запись в канал логов.

## Ранги

- Призванный
- Отказавшийся
- Принятый Обрядом
- Носитель Знака
- Хранитель Следа
- Собиратель Личин
- Проводник Лика
- Хранитель Печати
- Длань Обряда
- Голос Без Лица
- Первая Без Лица

Названия в `bot.py` должны полностью совпадать с названиями ролей Discord.

## Кто может использовать команду

- владелец сервера;
- пользователь с правом «Администратор»;
- роли «Владелец», «Администратор», «Первая Без Лица», «Голос Без Лица».

## Быстрый запуск на Windows

1. Установите Python 3.11 или новее.
2. Распакуйте архив.
3. В папке проекта откройте PowerShell.
4. Выполните:

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

5. Скопируйте `.env.example` в файл `.env`.
6. Вставьте в `.env` токен и ID сервера.
7. Запустите:

```powershell
python bot.py
```

## Настройка приложения Discord

В Discord Developer Portal:

1. Создайте приложение.
2. Откройте **Bot** и создайте бота.
3. Включите **Server Members Intent**.
4. Скопируйте токен в `.env`.
5. В **OAuth2 → URL Generator** отметьте:
   - `bot`
   - `applications.commands`
6. Для прав бота отметьте:
   - View Channels
   - Send Messages
   - Embed Links
   - Read Message History
   - Manage Roles
7. Откройте сгенерированную ссылку и добавьте бота на сервер.

После добавления переместите роль бота **выше всех рангов семьи**, которыми он должен управлять.

## Как получить ID сервера и каналов

1. Discord → Настройки пользователя → Расширенные → Режим разработчика.
2. ПКМ по серверу → Копировать ID сервера.
3. ПКМ по каналу → Копировать ID канала.

## Безопасность

Не публикуйте `.env` и никому не передавайте токен бота.
